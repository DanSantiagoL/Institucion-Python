¿Qué paquetes se deben instalar en visual studio Code?

Conector:

para MySQL:
pip install mysql-connector-python

para PostgreSQL:
pip install psycopg2

Paquete para ejecutar en la web:

pip install Flask

Paquetes para requests:

pip install requests
pip install lxml


Paquetes para Google auth y Google calendar

pip install --upgrade google-api-python-client google-auth-httplib2 google-auth-oauthlib
